# Palindrome radar code kata

Palindrome radar code kata see: https://www.codurance.com/katas/palindrome-radar

Done using [Test Driven Development](https://archive.org/details/est-driven-development-by-example/test-driven-development-by-example/) as recommended by [Codurance](https://www.codurance.com/)

## Getting started

Install [Node](https://nodejs.org/en/download) version v22.18.0 or above.

At the command line:

```console
npm run test
```

To run TDD tests.
